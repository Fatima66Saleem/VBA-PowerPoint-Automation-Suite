# Multi-Unit Converter (VBA)
## Project Overview
A comprehensive unit conversion tool built using VBA UserForms.

## Categories and Conversions
### Weight
Kilogram to Pounds: Kg × 1.75

### Temperature
Celsius to Fahrenheit: (C × 9/5) + 32

Celsius to Kelvin: C + 273.15

### Length
Kilometers to Meters: Km × 1000

### Time
Hours to Minutes: × 60

Hours to Seconds: × 3600

### Energy
Joules to Calories

Joules to Kilowatt-hours (kWh)

Joules to Electronvolts (eV)

## Technical Logic
Uses UserForm GUI with navigation buttons to switch between conversion frames

Implements Frame.Visible = True/False logic to create a clean, single-screen interface
