# Automated Result Card System (Built in PowerPoint)
## Project Overview
This project is an automated academic result generator built using VBA in PowerPoint.

## Logic & Features
Automatic Summation: Sums up marks for ICT, English, and Physics to calculate the total

### Pass/Fail Validation: Uses conditional logic to check if marks in all subjects are ≥ 40. If true, it displays "Pass", otherwise "Fail"

### Interactive UI: Users can enter marks and click the "ADD" button to see results instantly
